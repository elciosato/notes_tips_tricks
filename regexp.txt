. - Any one character
* - Match any number of previous character (including 0)
+ - Match any number of previous character
$ - End of line
^ - Beginning of line
\S - Any non-whitespace character
\s - Any whitespace character
? - optional
[a-z] - Any lowercase letter
[A-Z] - Any uppercase letter
[a-zA-Z] - Any letter
[0-9] - Any number

\ - escape character
